# 智行京都 · AI交通ダッシュボード

## 📖 プロジェクト概要
「智行京都 · AI交通ダッシュボード」は、**京都市の交通状況**を背景とした展示用可視化プロジェクトです。  
AI技術とスマート交通の理念を組み合わせ、大画面で道路の混雑状況、交通イベントを直感的に表示します。  
本プロジェクトは **展示・発表・教育** を目的としており、データは **静的JSON** を使用し、リアルタイムデータベースには依存しません。

---

## 🎯 プロジェクト背景
- 京都市中心部（四条通、河原町通、烏丸通など）は観光客と車両が集中し、特に休日に深刻な渋滞が発生。  
- 従来の監視・統計手法は全体の交通状況を直感的に把握しにくい。  
- 本プロジェクトは **UIを重視した大画面デザイン** により、「見える・わかる・伝わる」交通情報を実現する。  

---

## 🗺️ プロジェクト計画
### フェーズ1：準備と設計
- 表示エリアの決定（京都市主要道路・交差点）。  
- 静的JSONデータの準備。  
- システム構成とUIデザインガイドラインの策定。  

### フェーズ2：フロントエンド開発
- 地図ベース + 路線可視化（2D表示）。  
- KPIカード、渋滞ランキング、イベントタイムライン。  
- タイムライン再生機能（擬似リアルタイム）。  

### フェーズ3：最適化とデモ
- シナリオ切替、オートスライド。  
- アニメーションとインタラクションの改善。  
- 発表用シナリオの準備（例：全市ビュー → 交差点ビュー → イベント再生）。  

---

## 🛠️ 使用技術
- **フレームワーク**：React 18 + Vite + TypeScript  
- **地図ライブラリ**：Mapbox GL JS（ダークテーマ地図 + 道路ステータス表示）  
- **グラフライブラリ**：ECharts（KPI、トレンド、ランキング）  
- **スタイル**：Tailwind CSS（大画面対応、統一感のあるテーマ）  
- **データ形式**：静的JSON（スナップショット / イベント / タイムスライス）  

---

## 🎨 UIデザインガイドライン
- **基調**：ダークテーマ（背景 #0B1220）、未来感・テクノロジー感を演出。  
- **カラーパレット**：  
  - メイン：ブルー (#3B82F6)、シアン (#22D3EE)、パープル (#A78BFA)  
  - 状態色：  
    - 通行可能：グリーン #22C55E  
    - 渋滞：レッド #F43F5E  
    - 混雑：イエロー #FACC15  
- **レイアウト**：  
  - ヘッダー：タイトル、現在時刻、シナリオ切替ボタン  
  - メイン：地図（70%）+ サイドバー（イベント、ランキング）  
  - フッター：タイムライン再生コントローラ + KPI小型グラフ  
- **アニメーション**：  
  - 道路色のスムーズ変化（250–350ms）  
  - イベントポイントの波紋エフェクト（2.5s周期）  
  - 自動シナリオ切替時の地図フライアニメーション（900ms）
