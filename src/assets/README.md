# Assets フォルダ

静的リソースファイルを格納するディレクトリ

## 構成

```
assets/
├── images/       # 画像ファイル (.png, .jpg, .svg など)
└── icons/        # アイコンファイル (.svg, .ico など)
```

## 使用方法

```tsx
// 画像のインポート例
import logo from '@/assets/images/logo.png'

// SVGアイコンのインポート例
import iconTraffic from '@/assets/icons/traffic.svg'
```
